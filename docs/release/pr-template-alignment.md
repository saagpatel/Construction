# PR Template Alignment

The repository PR template is the required shape for release-ready review.

## Required Sections

- `What`
- `Why`
- `How`
- `Testing`
- `Performance impact`
- `Risk / Notes`

## Conditional Sections

- `Lockfile rationale` when lockfiles change
- `Baseline governance` when `.perf-baselines` changes

## Alignment Rule

A PR is not release-ready if required sections are missing or left empty.
