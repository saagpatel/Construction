# Day-1 Onboarding Checklist

## Access

- [ ] Repository access confirmed
- [ ] Local development prerequisites installed (Node, pnpm, Rust)
- [ ] Git identity configured

## Environment Setup

- [ ] `pnpm install` completed
- [ ] `cd src-tauri && cargo build` completed
- [ ] `pnpm dev:normal` starts successfully

## Quality Baseline

- [ ] `bash .codex/scripts/run_verify_commands.sh` passes locally
- [ ] Understand `.codex/verify.commands` ordering and purpose

## Workflow Readiness

- [ ] Branch naming convention understood: `codex/<type>/<slug>`
- [ ] Conventional commit format understood
- [ ] PR template expectations reviewed

## Operational Familiarity

- [ ] Read `docs/source-of-truth/README.md`
- [ ] Read release checklist and rollback runbook
- [ ] Know escalation owner for build/release blockers
